# homebrew-locklac

Homebrew tap for [lockLac](https://github.com/laurentlouk/locklac) — a macOS menu bar app that locks your screen while background tasks run.

## Install

```sh
brew tap laurentlouk/locklac
brew install locklac
```

## Usage

```sh
locklac           # launch menu bar app
locklac lock      # lock immediately
locklac --unlock  # unlock from SSH
```
